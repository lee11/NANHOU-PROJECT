Part of animation View In Hierarchy

An Animator contains a table of AnimationWrappers, keyed by prefix. The animator must be loaded from animations in a single directory sharing the same postfix.
Method __init__ Load an animator. This will automatically load wrapped animations, one for each prefix in the prefixList.
Method setFrameInterval This sets the length of an animation frame in game drawing frames (i.e. the number of game drawing frames that an single frame of the animation will be displayed for).
Method setAnimation Set a given animation to play, with a given frame interval. By default, it will also start at the first frame and loop continually after completing.
Method getCurrentImage Get the current image of the animator.
